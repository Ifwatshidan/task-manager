**Title**
 Student Task Manager System
 
**Description**
This project is a web-based task management system designed to help students organize and track their tasks efficiently. Users can add, edit, delete, and monitor their tasks through a simple and user-friendly interface. The system also includes a dashboard with visual data to show task progress.

**Features Included**
- Login system (basic validation)
- Dashboard with task summary and chart
- Add, edit, and delete tasks
- Search functionality
- Notes for each task
- Completed tasks page
- Dark mode toggle

**Instructions to Test Login**
- Enter any username and password to login
- After login, you will be redirected to the dashboard

**Framework / Libraries Used**
- HTML5  
- Bootsrap
- JavaScript (LocalStorage)  
- Chart.js
